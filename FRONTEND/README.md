# Frontend Contratos

Frontend em Vue 3 + TypeScript consumindo a API FastAPI em `http://127.0.0.1:8005/api/v1`.

## Estrutura

- `src/models`: contratos de dados
- `src/services`: acesso HTTP e integracao com a API
- `src/controllers`: estado e orquestracao da interface
- `src/views`: telas
- `src/components`: layout e componentes visuais

## Executar

```powershell
Set-Location d:\PROJETOINI\FRONTEND
npm install
npm run dev
```
